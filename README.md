# FreelanceHub - Full-Stack Freelance Platform

FreelanceHub is a fully functional, role-based freelance marketplace that connects clients with talented freelancers. Moving from a purely front-end design to a complete full-stack architecture, this platform features dynamic feeds, user authentication, side-by-side role tracking, and real-time application management.

## ✨ Key Features

### 🔐 Dual-Role Authentication
- Secure registration and login.
- Users choose their path: **Freelancer** or **Client**, unlocking unique dashboards tailored to their needs.

### 💻 For Freelancers
- **The Feed:** Browse through portfolios and project showcases uploaded by other freelancers.
- **Job Discovery Sidebar:** View active job proposals posted by clients, click to see full details, and filter by category.
- **Easy Application:** Submit proposals by sending a tailored pitch message and specifying a target budget.
- **Profile Management:** Edit personal details, upload new showcase projects, and track the live status of all submitted job applications.

### 💼 For Clients
- **Talent Discovery:** Browse freelancer projects with advanced filtering options to find the right talent.
- **Project Lifecycle Management:** Create, edit, and post new job proposals directly from the client profile.
- **Applicant Tracking:** View a list of freelancers who applied to your job postings with the power to **Accept** or **Reject** applications instantly.
